# 05  数据规范

1. 对于输入超过11位的数字会以科学计数的表达方式来显示

2. 已经录入的长串数字需要设置为文本格式后重新输入

3. 数字不计算，文本大法，早用早好。

   **开始 > 数字 > 文本**

4. 一横 (-) 一撇 (/) 年月日，任何符号输英文。

   **开始 > 数字 > 长日期**

5. 填写规范

   数据 > 数据验证 > 设置 > 验证条件**

   **数据 > 数据验证 > 输入信息** （温馨提示）

6. **Alt** + **↓** 快速录入

7. **文件 > 选项 > 高级 > 按 Enter 键后移动所选内容 > 方向**
